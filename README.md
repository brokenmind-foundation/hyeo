# hyeo-i386
  hyeo OS is an Operating System written in Assembly and C++
  It supports only i386 architecture.
  
# Features
  1. GRUB
  2. Global Descriptor Table
  3. Interrupt Descriptor Table
  4. Text-Mode Support
  5. System Calls
  6. Simple File System (HFS)
