# hyeo-i386
  hyeo OS is an Operating System written in Assembly and C++
  It supports only i386 architecture.
  
# Features
  1. Bootloader
  3. Global Descriptor Table
  4. Simple Paging
  5. Interrupt Descriptor Table
  6. IRQ's
  7. Text-Mode Support
  8. Syscalls
  9. Simple File System (HFS)
