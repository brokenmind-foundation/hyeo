# hyeo-i386
  hyeo OS is an Operating System written in Assembly and C++
  It supports only i386 architecture.
  
# Features
  - [x] GRUB
  - [x] Global Descriptor Table
  - [x] Interrupt Descriptor Table
  - [x] Text-Mode Support
  - [x] System Calls
  - [x] Simple File System (HFS)
